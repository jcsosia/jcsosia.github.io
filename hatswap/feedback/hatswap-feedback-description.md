# HatSwap Feedback
This page is dedicated to collecting feedback for the HatSwap project. We value your input and suggestions to help us improve the project and provide a better experience for our users.

## How to Provide Feedback
- Users can either submit a bug report, suggest a new feature, or provide general feedback.
- GitHub Issues Templates are used to streamline the feedback process and ensure that all necessary information is provided for each type of feedback.
- There should be appropriate buttons linked to the respective issue templates for easy access.
